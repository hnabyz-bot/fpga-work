# FPGA 이미지 데이터 수집 시스템

## 프로젝트 개요

FPGA 이미지 데이터 수집 시스템은 고성능 이미지 센서(ROIC)로부터 데이터를 수집하고 MIPI CSI-2 인터페이스로 출력하는 FPGA 기반 임베디드 시스템입니다. Xilinx Artix-7 FPGA를 기반으로 설계되었으며, 다중 채널 LVDS 인터페이스를 통해 TI ROIC 이미지 센서와 통신합니다.

## 핵심 기능

### 1. 다중 채널 LVDS 데이터 수신

- **14채널 고속 LVDS 인터페이스** (BLUE-HD)
- **12채널 LVDS 인터페이스** (BLUE, CYAN)
- 각 채널 독립적 비트 정렬 및 디시리얼라이저
- 프레임 클록 동기화 및 데이터 재정렬
- 2.34 MHz 데이터 클록 지원

### 2. MIPI CSI-2 출력 인터페이스

- **4-lane MIPI CSI-2** 고속 출력
- HS(High Speed) 및 LP(Low Power) 모드 지원
- AXI-Stream 기반 데이터 전송
- 실시간 이미지 스트리밍 지원

### 3. 시퀀스 제어 시스템

- **FSM 기반 이미지 획득 시퀀스 제어**
- LUT 기반 프로그래머블 시퀀스
- 리셋, 대기, 바이어스, 플러시, 노출, 리드아웃 상태
- AED(Auto Exposure Detection) 모드 지원

### 4. 유연한 구성 인터페이스

- **SPI 슬레이브** 기반 레지스터 맵
- 런타임 ROIC 파라미터 설정
- 상태 모니터링 및 진단
- I2C 마스터를 통한 외부 디바이스 제어

### 5. 게이트 드라이브 제어

- **8채널 XAO 출력** 신호 생성
- **8채널 STV_LR 출력** 제어
- CPV 및 OE 제어 신호
- 정밀한 타이밍 제어

### 6. 고급 시뮬레이션 환경 (NEW)

- **Questa Advanced Simulator** 지원
- 완전한 테스트벤치 프레임워크
- 회귀 테스트 자동화
- GUI 및 배치 모드 지원

## 프로젝트 변형 비교

본 프로젝트는 세 가지 하드웨어 변형을 지원합니다:

| 특성 | BLUE-HD | BLUE | CYAN |
|------|---------|------|------|
| **설명** | HD 버전 (주력 개발) | GEN2 레거시 버전 | Wired-only TI ROIC |
| **LVDS 채널** | 14채널 | 12채널 | 12채널 |
| **타겟 보드** | BLUE-HD 보드 | BLUE 보드 | CYAN 보드 |
| **FPGA 디바이스** | xc7a35tfgg484 | xc7a35tfgg484 | xc7a35tfgg484 |
| **시뮬레이터** | Questa 2024.3 | Vivado Simulator | Vivado Simulator |
| **주요 용도** | 고해상도 이미지 수집 | 범용 이미지 수집 | 유선 전용 적용 |
| **개발 상태** | 활성 개발 | 유지보수 | 특수 용도 |

### BLUE-HD-FPGA (주력 프로젝트)

- 최신 14채널 LVDS 인터페이스
- 확장된 게이트 드라이브 제어 (8채널 XAO, 8채널 STV_LR)
- 완전한 Questa 시뮬레이션 환경
- 포괄적인 테스트벤치 포함 (10개 이상)
- ROIC SPI 단일 신호 구성

### BLUE_FPGA (레거시)

- GEN2 프로젝트용 12채널 구성
- 기본 LVDS 인터페이스
- Vivado Simulator 기반 시뮬레이션
- BLUE-HD의 기반 아키텍처

### CYAN-FPGA (특수용도)

- TI ROIC 전용 유선 인터페이스
- BLUE 파생 버전
- 간소화된 인터페이스 구성
